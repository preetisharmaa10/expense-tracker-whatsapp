 Expense Tracker with WhatsApp Automation
A Python-based desktop utility that records daily expenses and automatically sends WhatsApp confirmations and daily summaries.
  
  Features
- Add and store daily expenses
- WhatsApp confirmation after adding expense
- Daily summary message
- Category-wise totals
- Simple file storage
- 
 Tech Stack
- Python
- pywhatkit
- File handling
- Automation scripting

 How to Run

Install dependency:
pip install pywhatkit

Run expense logger:
python expense_whatsapp.py

Run daily summary:
python daily_summary.py
